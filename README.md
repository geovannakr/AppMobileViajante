# App Viajantes

O **App Viajantes** é um aplicativo multiplataforma desenvolvido principalmente com **Flutter**, integrando uma combinação de várias linguagens e tecnologias para garantir um desempenho robusto e uma experiência de usuário fluida e agradável.

## Tecnologias Utilizadas

O projeto faz uso de uma combinação de tecnologias nativas e **Flutter**, criando uma aplicação que atende a diferentes plataformas com eficiência e flexibilidade:

- **C++ (45.1%) & CMake (36.9%)**: Responsáveis pelo núcleo nativo e pela integração de funcionalidades que tornam o app mais **performático**.
- **Dart (9.1%)**: Base da lógica e da interface do **Flutter**, proporcionando uma navegação ágil e interativa.
- **Swift & C**: Garantem a **compatibilidade com o iOS** e funções específicas de baixo nível.
- **HTML e outros recursos**: Enriquecem os elementos visuais e funcionais, melhorando a experiência do usuário.

---

## Funcionalidades Principais

O aplicativo oferece uma série de funcionalidades úteis e intuitivas, proporcionando uma experiência completa ao usuário:

- **Login Simples**: O usuário pode criar e acessar seu perfil de forma prática e direta.
  
- **Cadastro de País Favorito**: Permite ao usuário selecionar e salvar seu **país favorito** no aplicativo, facilitando a personalização da experiência.

- **Visualização Personalizada**: Ao selecionar o país, o app exibe automaticamente a **bandeira correspondente** de maneira interativa, oferecendo uma experiência visualmente envolvente e dinâmica.

---

## Por Que É Especial?

Este projeto destaca-se pela **integração de múltiplas linguagens** e **tecnologias nativas**, permitindo que o aplicativo tenha um excelente **desempenho** e uma **interface fluida**. A combinação de **Flutter** com **C++**, **Swift**, e **C** possibilita uma experiência robusta e otimizada para diferentes plataformas, enquanto a personalização visual e interatividade criam um app não só funcional, mas também **visualmente atraente**.

O **App Viajantes** é uma excelente amostra de como é possível combinar **tecnologias nativas e multiplataforma** de maneira eficaz e elegante.

## Como Executar

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/usuario/app-viajantes.git

2. **Abra o projeto** no seu editor Flutter preferido (VS Code, Android Studio, etc.).

3. **Instale as dependências**:

   ```bash
   flutter pub get
   ```

4. **Execute o app**:

   ```bash
   flutter run
   ```

5. O app estará disponível no emulador ou dispositivo conectado para navegação e testes.
